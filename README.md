# opengemini-mate

## usage

| Env                 | Explain                                                                   |
|---------------------|---------------------------------------------------------------------------|
| `DEPLOY_TYPE`       | deploy type, current support `stand-alone`                                |
| `META_HTTPS_ENABLE` | enable ts-meta server https, default value `false`, support `true\|false` | 
| `HTTPS_ENABLE`      | enable ts-sql server https, default value `false`, support `true\|false`  | 
| `HTTPS_CERTIFICATE` | after enable https, the certificate path, default value `""`              |
| `HTTPS_PRIVATE_KEY` | after enable https, the private key path, default value `""`              |
